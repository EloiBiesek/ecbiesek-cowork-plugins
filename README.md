# ECBIESEK Cowork Plugins

Marketplace de plugins da **ECBIESEK** para [Claude Cowork](https://claude.com).

## Como usar

No Claude Desktop/Cowork:
1. Abrir **Navegar por plugins**
2. Clicar em **+** → **Adicionar marketplace do GitHub**
3. Digitar: `ecbiesek/ecbiesek-cowork-plugins`

No Claude Code (CLI):
```bash
/plugin marketplace add ecbiesek/ecbiesek-cowork-plugins
/plugin install gestao-empreiteiros@ecbiesek-cowork-plugins
```

## Plugins disponíveis

| Plugin | Descrição |
|--------|-----------|
| **gestao-empreiteiros** | Extração automatizada de SEFIP e NFSe de empreiteiros de obras de construção civil |

## Sobre

Plugins desenvolvidos pela **INOTEC** (braço de tecnologia da ECBIESEK) para automação de processos de construção civil.
